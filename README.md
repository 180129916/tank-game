# 🎮 坦克大战游戏

这是一个使用HTML5 Canvas和JavaScript制作的经典坦克大战游戏。

## 📸 游戏预览

游戏界面简洁直观，支持键盘操作，具有完整的碰撞检测和计分系统。

## 🎮 游戏玩法

### 控制方式
- **W** - 向上移动
- **A** - 向左移动  
- **S** - 向下移动
- **D** - 向右移动
- **空格键** - 发射子弹

### 游戏规则
- 控制你的绿色坦克躲避敌人攻击
- 射击红色敌人获得分数
- 每个敌人被击中可得10分
- 玩家有3点生命值，被子弹击中或与敌人碰撞会减少生命值
- 生命值为0时游戏结束
- 按**R键**重新开始游戏

## 🛠️ 技术特点

- **纯前端实现** - 无需后端，所有逻辑都在浏览器中运行
- **Canvas渲染** - 使用HTML5 Canvas API进行游戏渲染
- **面向对象设计** - 清晰的代码结构，易于扩展
- **实时碰撞检测** - 精确的物理碰撞系统
- **响应式设计** - 支持不同屏幕尺寸

## 🚀 快速开始

1. 克隆仓库到本地：
```bash
git clone https://github.com/180129916/tank-game.git
```

2. 进入项目目录：
```bash
cd tank-game
```

3. 直接在浏览器中打开`tank-game.html`文件即可开始游戏！
